== Description ==

The Article Voting Plugin allows website visitors to vote on various articles with a simple Yes or No option. It provides a user-friendly and efficient voting system integrated seamlessly with WordPress websites.

== Installation ==

1. Upload the `article-voting` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Configuration ==

1. Once activated, the plugin will automatically add the voting feature at the end of every single post article.

== Features ==

- Simple Yes or No voting system.
- AJAX-based submission for seamless user experience.
- Prevents duplicate voting by tracking user IP addresses.
- Automatic display of voting feature at the end of single post articles.
- Admin widget to view voting results for each post.

== Frequently Asked Questions ==

1. Can users change their votes after submitting?
   - No, once a user has submitted their vote, it cannot be changed.

2. How does the plugin prevent duplicate voting?
   - The plugin tracks user IP addresses to prevent multiple votes from the same user.

3. Can I customize the appearance of the voting feature?
   - Yes, you can customize the voting buttons and appearance in the plugin settings.
